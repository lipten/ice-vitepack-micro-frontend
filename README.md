## Getting Started

```
git clone git@github.com:lipten/ice-vitepack-micro-frontend.git

cd ice-vitepack-micro-frontend
```

* base-app
```
cd base-app
yarn install
yarn start:micro
```

* app1
```
cd app1
yarn install
yarn start:micro
```

* app2
```
cd app2
yarn install
yarn start:micro
```

访问[http://localhost:6010](http://localhost:6010)
